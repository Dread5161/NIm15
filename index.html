Правила игры

Цель игры - снять все палочки, следуя правилам. Игроки по очереди удаляют палочки из одного из трех рядов. Игрок может удалить любое количество палочек из одного ряда, но не более 20 палочек за раз.

  • 1. Каждый игрок по очереди удаляет палочки из одного из рядов.
  • 2. В один ход можно удалить не более 20 палочек из одного ряда.
  • 3. Игроки могут выбирать ряд, в котором они хотят удалить палочки.
  • 4. Игра заканчивается, когда все палочки будут сняты.
  • 5. Побеждает тот, кто оставит последнюю палочку противнику.
